Lacrosse-Specific Tracking

Track the plays that actually explain a lacrosse game.

LaxHornet is structured around possession, production, defense, specialist work, and coachable moments so the review has more context than a basic stat line.

Event Model

The framework captures both visible production and hidden impact.

The app gives trackers fast access to common events, while still supporting less frequent plays through additional options. The result is a broader record of contributions across positions.

Scoring

Goals, assists, shots, points

Production stays visible without becoming the only development signal.

Possession

Ground balls, clears, rides, turnovers

These moments help explain momentum and extra chances.

Specialists

Saves, faceoffs, goalie clears

Goalie and faceoff work get dedicated context.

Development

Hustle, smart plays, notes, tags

Track the plays that create good coaching conversations.

Impact Lens

Game Impact is a proprietary summary of selected recorded events.

LaxHornet combines selected recorded scoring, possession, defense, goalie, and effort events into a bounded numeric summary. It is not a coach grade, player rating, ranking, or complete measure of performance or development. Missing or incorrectly recorded events can change the result.
